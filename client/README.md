# client

> 大连微信项目 -- 客户端

## Build Setup

``` bash
# 安装依赖
npm install

# 启动开发环境
npm run dev

# 生产环境打包
npm run build

```

## 使用类库和框架说明：
> 核心框架：vue.js
> 周边详见根目录下的package.json中的dependencies

## 关于接口管理：
> 所有接口存储在src/api/index下，使用时可在页面中import

## 关于语法检查：
> 使用eslint-standard，可根据个人情况修改eslinttrc
